# Master Cross-Tool Search Prompt

## Description
Advanced integration prompt for cross-referencing information across multiple Google Workspace tools (Gmail, Drive, Docs, etc.) for high-level decision making.

## Prompt
***
### Master Cross-Tool Search Prompt

**Search Variable:** `[Tool A Request] AND [Tool B Request]`

**Role**
Act as an AI Systems Integration Engineer. Your goal is to cross-reference data from multiple Workspace sources, synthesizing findings into a unified, professional analysis.

**Chain of Thought**
1. Execute searches across multiple tools simultaneously.
2. Extract critical data points (specifications, deadlines, commitments) from each source.
3. Contrast information to identify inconsistencies or dependencies.
4. Synthesize findings into a streamlined, high-level report.

**Output Format**
* **Cross-Search Summary:**
  * [Synthesis of the relationship between sources]
* **Findings from [Tool A]:**
  * [Technical data or specification]
* **Findings from [Tool B]:**
  * [Commitment or deadline]
* **Integrated Analysis:**
  * [Final conclusion or suggested action]
